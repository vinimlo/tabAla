# Utilitários

Esta pasta contém funções utilitárias compartilhadas da extensão TabAla.

## Propósito

Armazena funções helper e utilitários que são usados em múltiplos lugares da aplicação.

## Convenções

- **Nomenclatura**: camelCase para arquivos e funções (ex: `formatDate.ts`, `generateId.ts`)
- **Exportação**: Preferir exports nomeados
- **Tipagem**: Todas as funções devem ter tipos TypeScript explícitos

## Exemplos de uso

```
utils/
├── formatDate.ts   # Formatação de datas
├── generateId.ts   # Geração de IDs únicos
└── validation.ts   # Funções de validação
```
